
# My Restaurant List

Final project for Dicoding's Front-End Web Development course.  

Link: https://my-restaurant-list.netlify.app/
## Run Locally

Clone the project

```bash
  git clone https://github.com/anis000romzi/my-restaurant-list.git
```

Install dependencies

```bash
  npm install
```

Build the app

```bash
  npm run build
```

Start the server

```bash
  npm run serve
```
